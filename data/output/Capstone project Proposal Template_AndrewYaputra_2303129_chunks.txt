Project Objectives:

(Please use a separate sheet if necessary)

Implement a complete oral assessment workflow: 1. Start, 2. Record, 3. Automatic Speech Recognition (ASR), 4. Reasoning, 5. Avatar Response, 6. Feedback

Develop a system using pre-designed examiner prompts and configurable avatars.

Provide automated, rubric-based feedback based on evidence from the session transcript.

Project Scope:

(Please use a separate sheet if necessary)

In-Scope: Paper ingestion (PDF/Text), audio recording/ASR, agent-based dialogue orchestration, avatar speech output, and rubric-based feedback generation.

Out-of-Scope: Custom avatar training, general knowledge questions outside the provided paper, and biometric monitoring

Project Outputs and Deliverables

(Please use a separate sheet if necessary)

Functional Product: A system supporting end-to-end oral assessment.

Documentation: Technical specifications of the 6-agent architecture (Session Orchestrator, RAG, Dialogue Manager, etc.).

Evaluation Report: Results from technical testing and pilot evaluations (consistency checks and student surveys).
